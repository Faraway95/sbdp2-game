# 美术设计文档 (Art Bible): 《上班打牌！肉鸽版》
文档版本: 1.0
核心风格: 精致荒谬 (Refined Absurdity)
指导原则: 用设计咨询公司的方法论，去包装一个草台班子的闹剧。
### 1. 视觉风格总览 (Visual Style Overview)
本作的视觉核心是“精致荒谬”，旨在通过专业、高级的视觉语言与荒诞、戏谑的职场内容形成强烈反差，从而创造出独特的黑色幽默体验。我们的目标是让玩家在看到界面的第一眼，感受到的是大型科技公司或咨询报告般的专业性，但在理解内容后，又能会心一笑，领悟其讽刺本质。
#### 1.1 核心氛围
整体氛围是压抑的专业感与失控的混乱感的结合体。视觉上，我们追求极致的整洁、秩序和现代感，仿佛一切尽在掌握。但在内容和动画细节上，则要处处流露出角色内心的疲惫、抓狂与崩溃。这种视觉与情感的割裂是所有设计的出发点。
#### 1.2 色彩规范 (Color Palette)
色彩系统分为基础色板和功能色板。基础色板奠定“精致”的基调，功能色板则用于凸显“荒诞”的玩法元素。
| 色彩类别 | 颜色 | HEX | 用途说明 |
| :--- | :--- | :--- | :--- |
| 主品牌色 | 企业深蓝 | #1E3A8A | UI主背景、标题栏、核心面板、专业/科技主题元素。营造沉稳、可靠的氛围。 |
| 辅助品牌色 | 高级中性灰 | #4B5563 | UI次级背景、正文文本、未激活按钮、分隔线。提供视觉层次，强化专业感。 |
| 高亮/行动色 | 活力蓝 | #3B82F6 | 可交互按钮、链接、玩家回合高亮。引导用户操作。 |
| KPI警戒红 | 警戒红 | #EF4444 | 伤害数值、负面效果、生命值警报、错误提示。代表“业绩压力”和危险。 |
| “画饼”荧光绿 | 荧光绿 | #34D399 | 增益效果、临时资源增长、市场/营销主题卡牌。代表“虚假繁荣”和机遇。 |
| 资金警告黄 | 警告黄 | #FBBF24 | 经费、金钱、人力资源主题卡牌、警告提示。代表资源和潜在风险。 |
| “心理防线”蓝 | 防护蓝 | #60A5FA | 格挡值、护盾、防御性技能。代表冷静和保护。 |
#### 1.3 字体规范 (Typography)
字体的选择严格区分“表面工作”与“内心独白”。
| 字体类别 | 字体名称 | 用途 | 设计意图 |
| :--- | :--- | :--- |
| 界面主字体 | 思源黑体 (Source Han Sans) | 标题 (Bold)、正文 (Regular)、数值、按钮文本 | 现代、清晰、专业。强化UI作为“项目管理软件”的严肃感。 |
| 风味文本字体 | 站酷快乐体 / 类似的手写体 | 卡牌、遗物、事件的“风味文本” (Flavor Text) | 潦草、个人化。像是员工在专业文档下的吐槽和批注，是打破“精致”外壳、暴露“荒谬”内核的点睛之笔。 |

### 6. 视觉效果 (VFX)
设计原则: VFX必须是符号化、信息化、UI化的。避免使用火、电、爆炸等传统特效，转而使用更符合“项目管理”主题的动态图形 (Motion Graphics)。
| 动作名称 | VFX 概念描述 | 声音效果 (SFX) |
| :--- | :--- | :--- |
| 画饼 | 从使用者身上绽放出一个明亮的绿色饼状图，迅速扩大并覆盖屏幕，图上的百分比疯狂跳动。 | 激励人心的PPT背景音乐，但结尾突然变调。 |
| 甩锅 | 一条锐利的红色矢量箭头从使用者身上射出，击中目标后，目标身上出现一个红色的“责任”标签或Bug图标。 | 清脆的“丢出”声，和Jira分配任务的提示音。 |
| 开会 | 屏幕上弹出一个巨大的、半透明的日历邀请窗口，将所有角色框住。窗口内的角色动画变慢，呈呆滞状。 | 单调的电话会议等待音乐，或某人忘记静音的键盘敲击声。 |
| 业绩压力 (伤害) | 目标身上跳出红色的伤害数字，字体为思源黑体。数字落下后会像像素一样碎裂。 | 尖锐的、类似系统报错的提示音。 |
| 心理防线 (格挡) | 使用者身前出现一个由蓝色六边形网格构成的、半透明的数据护盾。受到攻击时，护盾上会出现涟漪。 | 防火墙启动的电子音效。 |
| 需求变更 | 对目标进行快速、多次的低伤害攻击。视觉上表现为无数个小小的、红色的“编辑”或“修改”图标像机关枪一样射向目标。 | 快速、密集的打字声和系统提示音。 |
